# מלבלב — הגדרת Google Sheets לניהול תוכן

## הוראות הקמה

1. צרו Google Sheet חדש בשם **"מלבלב — ניהול תוכן"**
2. צרו את הטאבים (sheets) המפורטים למטה
3. בכל טאב, השורה הראשונה היא כותרת: `key | value_he | value_en`
4. שתפו את הגיליון: **Share → Anyone with link → Viewer**
5. העתיקו את ה-Spreadsheet ID מה-URL (החלק בין `/d/` ו-`/edit`)
